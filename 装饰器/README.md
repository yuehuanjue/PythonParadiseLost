

## 简介:
Python 对某个对象是否能通过装饰器（ @decorator）形式使用只有一个要求：decorator 必须是一个“可被调用（callable）的对象。
callable 对象的种类：
1、函数
2、带 __call__ 的 类
3、偏函数
